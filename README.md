# Pattern-extender
Simple program to extend repeat patterns.

## Usage
```bash
pextender [input image path] [output image path] [output image height (pixels)] [output image width (pixels)]
```
**Example usage**
```bash
pextender pattern.jpeg repeatpattern.jpeg 480 980
```
